### Key Agent & CLI Capabilities
- **Agent Mode:** Performs multi-step complex workflows, manages sessions, and delegates tasks.
- **Edit Mode:** Performs multi-file edits in place using natural language instruction.
- **Autopilot CLI Mode:** Executes multi-step command sequences autonomously without per-step human approval.
- **Sub-Agents:** specialized sub-processes for: *Explore*, *Task*, *Code Review*, and *Plan*.
- **Model Context Protocol (MCP):** Connects Copilot directly to external databases, files, and tools without passing massive context tokens.

### Context Exclusions & Safeguards
- **`.copilotignore` file:** Used to filter out specific directories or files from being used as context by Copilot.
- **Toxicity Filter:** Prevents generating toxic, hateful, or discriminatory suggestions.
- **Duplication Detection Filter:** Scans generated suggestions against GitHub's public code database, blocking exact matches if enabled.
